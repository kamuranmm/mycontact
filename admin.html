Domain İdarəsi (White-label)
Şirkətlər üçün öz domeni ilə white-label xidmət. Hər domenə limitli sub-admin icazəsi verin.
ℹ️ Necə işləyir?
1️⃣ Şirkət alır → onlara domain əlavə edirsən (məs. cards.sirket.az)
2️⃣ DNS yazısı → şirkət domain qeydiyyatçısında CNAME əlavə edir
3️⃣ Sub-admin icazəsi → şirkətin öz adminini seçirsən, yalnız öz domeninin istifadəçilərini idarə edə bilir
4️⃣ Sən master admin olaraq hər şeyi görürsən — plan, limit, billing sənin əlindədir
🌐 Subdomain STANDARD
brand.apps.mycontact.az — avtomatik aktivdir
brand.apps.vizitaz.az/profil
🔗 Custom Domain PRO/BUSINESS
brand.com — DNS ilə qoşulur
cards.brand.com/profil

Sub-admin yalnız bu domenin istifadəçilərini görə bilər (profil açmaq, şablon dəyişmək). Plan/billing idarəsi yalnız sizin əlinizdədir.

📋 DNS Konfiqurasiya Rəhbəri
Domain qeydiyyatçınızda (Aznet, NIC.az, GoDaddy, Cloudflare vs.) aşağıdakı DNS qeydlərini əlavə edin:
CNAME qeydi (subdomain üçün):
Host: cards (və ya istənilən subdomain)
Value: proxy.apps.mycontact.az
A qeydi (kök domain üçün):
Host: @
Value: 76.76.21.21
TXT qeydi (doğrulama):
Host: _cardpro
Value: verify=[sistem tərəfindən verilir]
⏱ DNS yayılması 24-48 saat çəkə bilər. Status avtomatik yenilənir.
