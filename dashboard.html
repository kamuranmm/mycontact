CRM İnteqrasiya
Make.com vasitəsilə HubSpot, Salesforce, Pipedrive, Zoho və digər CRM-lərə qoşulun.
Make.com Webhook URL
Make.com-da scenario yaradın, webhook URL-i alın və aşağıya yapışdırın.
Düzgün webhook URL daxil edin
Hansı CRM-ə qoşmaq istəyirsiniz?
1
make.com-da qeydiyyat edin → Create a scenario
2
Trigger: Webhooks → Custom webhook → Add → URL kopyalayın, yuxarıya yapışdırın, Saxla
3
Test et düyməsinə basın — Make-də data gəlsin
4
Make-də + → HubSpot CRM → Create or Update a Contact
5
HubSpot-a qoşun → Properties → Map toggle ON edin → sahələri birləşdirin:
firstname=name  |  email=email  |  phone=phone  |  company=company
6
Scenario-nu ON edin → Save ✓
